# Coffee Shop Domain Model
## Overview
This Python project models is about a coffee shop using object-oriented programming principles
It demonstrates about the relationship between coffes,cutsomers and orders

The project follows a real-world situations:
 - A customer can place multiple orders.
 - A coffee can be orderd by multiple customers.
 - An order belongs to a single customer and a single coffee.
 - The many-to-many relationship between Customers and Coffees is handled through Orders
